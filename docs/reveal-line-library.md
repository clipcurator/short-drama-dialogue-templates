# Reveal Line Library

Reveal lines should change what the audience understands.

## Identity reveal

```text
You are not looking for the heir. You are looking at her.
```

## Relationship reveal

```text
He is not my enemy. He is my brother.
```

## Secret reveal

```text
The accident was not an accident.
```

## Evidence reveal

```text
Then explain why your signature is on this.
```

## Cliffhanger reveal

```text
Before you leave, you should know who sent that message.
```

## Review questions

- Does the reveal change the power dynamic?
- Does it create a new question?
- Does it connect to earlier setup?
- Is the line short enough for a mobile scene?
