# Conflict Dialogue Patterns

## Accusation

```text
You said [claim], but [evidence] proves [truth].
```

Use when one character confronts another with evidence.

## Boundary

```text
I can lose [thing], but I will not give you [boundary].
```

Use when a character refuses pressure.

## Power reversal

```text
You thought [power position], but [new fact] means [reversal].
```

Use when status changes in the scene.

## Emotional wound

```text
It was never about [surface issue]. It was about [hidden pain].
```

Use when a scene shifts from argument to emotional truth.

## Threat

```text
If you [action], everyone will know [secret].
```

Use when stakes need to rise quickly.
