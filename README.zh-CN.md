# 短剧对白模板库

短剧对白必须推进得很快：它要制造冲突、暴露秘密、定义关系，并给观众继续看下一集的理由。本仓库整理竖屏短剧、微短剧、AI 电影和 script-to-video 工作流中的可复用对白模板。

它适合需要把对白直接推进到剧本、分镜和视频生成的编剧与 AI 生产团队。

## 为什么对白重要

短剧里的对白通常承担比传统场景更多的叙事任务：

- 快速引入冲突。
- 解释人物关系张力。
- 揭示秘密。
- 制造情绪转折。
- 设置集尾悬念。

对白弱，即使设定很好，单集也容易显得平。

## 对白模式

| 模式 | 适合场景 | 示例框架 |
|---|---|---|
| 指控 | 开场直接制造冲突 | “你早就知道真相，却还是骗了我。” |
| 拒绝 | 表现权力或边界 | “这份协议，我不会签。” |
| 揭示 | 改变观众理解 | “那个孩子，不是他的。” |
| 反转 | 一句话翻转地位 | “不是你开除我，是我买下了这家公司。” |
| 悬念 | 集尾留下开放问题 | “那你告诉我，为什么文件上有你的名字？” |

## 场景结构

1. 从紧张开始，不从寒暄开始。
2. 每个角色都要有目标。
3. 让对白体现权力变化。
4. 用短句保持移动端节奏。
5. 结尾用问题、威胁、揭示或决定收束。
